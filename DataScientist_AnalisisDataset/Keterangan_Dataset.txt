==========================================================
DATASET CAPSTONE CC26-PSU014 - ANALISIS PERSEBARAN SPPG
==========================================================

Folder Google Drive berikut berisi seluruh berkas yang digunakan dalam proses pengerjaan proyek Capstone CC26-PSU014 terkait analisis persebaran Satuan Pelayanan Pemenuhan Gizi (SPPG) dan data pendidikan.

Isi folder meliputi:
1. DATASET AWAL (RAW DATA)
   - Dataset hasil scraping dari dua sumber data.
   - Data mentah yang belum melalui proses pembersihan maupun transformasi.

2. NOTEBOOK PENGOLAHAN DATA (.ipynb)
   - Berisi tahapan pengolahan data secara lengkap.
   - Mencakup proses scraping, cleaning, transformasi, integrasi dataset, analisis, hingga pembuatan dataset akhir.

3. HASIL PENGOLAHAN DATA (.csv)
   - Dataset hasil akhir yang telah dibersihkan dan diproses.
   - Tersedia dalam folder ini meskipun sebagian file hasil juga disertakan pada repository GitHub.

4. LAPORAN TEKNIS KOMPREHENSIF
   - Dokumentasi lengkap proyek.
   - Menjelaskan latar belakang, sumber data, metodologi, tahapan pengolahan data, analisis, visualisasi, hingga kesimpulan.

Referensi Sumber Data:
- SPPG Operasional - Badan Gizi Nasional
  (digunakan untuk memperoleh informasi persebaran SPPG berdasarkan wilayah)
- Portal Data Pendidikan - Kemendikdasmen
  (digunakan untuk memperoleh data pendidikan sebagai dasar analisis kebutuhan layanan)
